# Jesse-Agent

A personal AI agent built from scratch in TypeScript. No frameworks, no SDK magic — just a clean agentic loop with full understanding of every line.

## What is this?

This is Jesse's personal AI agent project. The goal is to learn agent architecture by building one from zero, then gradually evolve it into a daily-use tool.

## Tech Stack

- **Language:** TypeScript + Node.js
- **LLM:** OpenAI API (GPT-4o)
- **Architecture:** Custom agentic loop (ReAct pattern)
- **Framework:** None — built from scratch for learning
